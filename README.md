# csv2xyd-NDMOutAnalyzer
<img width="140" height="120" src="https://github.com/NCBC-Lab/csv2xyd-NDMOutAnalyzer/blob/main/images/NDMOutAnalyzer.png?raw=true">

NDMoutAnalyzer is a Python companion tool module for csv2xyd that automates parsing and tabulating technical .out files from NDM/vNDM. It filters results by score, manages taxa, and computes spatial overlaps using beta diversity, generating publication-ready mosaic consensus maps.

<p><strong>Key Capabilities:</strong></p>

Automated Data Parsing: Effortlessly converts technical .out files into structured, tabular formats (Excel/CSV), saving hours of manual data extraction.

Intelligent Filtering: Refine your analysis by filtering Areas of Endemism (AoE) based on specific score thresholds and taxonomic groups.

Spatial Analysis: Integrates seamlessly with your geo-spatial datasets to compute overlap metrics and beta diversity indices, quantifying relationships between consensus areas.

Publication-Ready Visualization: Automates the generation of mosaic consensus maps, designed to communicate biogeographic patterns clearly in academic publications.

The new module: Why NDMOutAnalyzer
Designed for high-performance biogeography, this tool eliminates the technical bottlenecks associated with large-scale endemism studies, allowing researchers to focus on biological interpretation and hypothesis testing rather than data formatting.

1. <p><strong>Analyze NDM/vNDM Output (.out):</strong></p>

This option opens the NDM/vNDM Output Analyzer module, a dedicated interface for processing one or multiple .out files generated by NDM or vNDM.
Main functionalities include:

  •	Loading multiple .out files corresponding to different grid sizes or analytical runs.

  •	Filtering endemic species by minimum and maximum endemicity score.

  •	Optional inclusion or exclusion of supraspecific categories (GRP- prefix).

  •	Automatic extraction and summary of:

    o	Individual endemic areas (SET NR.)

    o	Consensus areas

    o	Species represented in each analysis

    o	Endemic species per area and per consensus

  •	Export of all results to a multi-sheet Excel file, facilitating downstream analyses and reporting.

2. <p><strong>Geospatial Consensus Analysis:</strong></p>
This option performs a spatial analysis of consensus areas, based on polygon (WKT files) and point files previously exported from NDM/vNDM analyses. The analysis requires folders containing paired files in CSV format:

  •	conX_N.csv → polygon representation of consensus areas.

  •	pointX_N.csv → species occurrence points associated with each consensus.

Each pair corresponds to a consensus area X derived from N individual areas.

Main outputs include:

  •	Consensus summary: Area size, number of species, polygon area, and grid resolution.

  •	Consensus overlap analysis: Pairwise spatial overlap between consensus areas, including intersection area and Jaccard index.

  •	Shared species and genera: Identification of shared and exclusive species and genera among consensus areas across different grid sizes.

  •	Beta Diversity Partitioning (Baselga, 2009): Advanced decomposition of biotic dissimilarity between areas, calculating:

    o	Turnover (βsim): Species replacement between areas.

    o	Nestedness (βsne): Dissimilarity resulting from species loss/gain (subsets).

    o	Total Dissimilarity (βsor and βjac): Overall taxonomic difference.

All results are exported to a single Excel file with multiple sheets (including specialized sheets for genus-level summaries and analytical statistics), enabling comparative spatial analyses and integration with GIS software.

3. <p><strong>Export Consensus Mosaics:</strong></p>
This module provides a interface for creating high-quality, multi-panel figures for publication. It allows for the visual comparison of consensus areas across different resolutions or treatments.
Main functionalities include:

  •	GIS Base map integration: Allows the loading of external vector layers (Shapefile .shp or GeoPackage .gpkg) to provide geographic context such as political borders, ecoregions, or topography.

  •	Automated grid visualization: Generates a synchronized mosaic where each panel displays a specific consensus area. The system ensures spatial consistency by maintaining the same geographic extent and scale across all panels.

  •	Publication-Ready export: Exports the final composition (mosaic) as a high-resolution PNG file (300 DPI), suitable for scientific journals.
